# تطبيق تعلم اللغة الإيزيدية (A1) | Ezidi Language Learning App

تطبيق ويب تفاعلي مصمم بأسلوب **Duolingo** لتعليم أساسيات اللغة الإيزيدية للمستوى A1.

### 🌟 المميزات:
- يدعم 3 لغات أم (العربية، الألمانية، الإنجليزية).
- نظام تفاعلي متكامل (شريط تقدم، 5 قلوب فرص، واجهات ملونة وتغذية راجعة فورية).
- متوافق بالكامل للعمل على الهواتف والكمبيوتر.
